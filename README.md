# web-pos-system — Demo POS & Lunch LK

Демонстрационный прототип кассовой системы и личного кабинета для заказа обедов предприятия.  
База данных: **Cloud Firestore (Spark Plan)** — плоская структура коллекций.

## Быстрый старт

```bash
npm install

# Запуск модулей (каждый на своём порту)
npm run dev:lk        # Личный кабинет       → http://localhost:3001
npm run dev:admin     # Админ-панель         → http://localhost:3002
npm run dev:kitchen   # Кухонный терминал   → http://localhost:3003
npm run dev:delivery  # Терминал выдачи     → http://localhost:3004
npm run dev:queue     # Экран очереди       → http://localhost:3005
```

## Структура проекта

```
web-pos-system/
├── shared/
│   ├── firebase.js      # Инициализация Firebase (Auth + Firestore)
│   ├── schema.js        # Схема коллекций + фабричные функции документов
│   ├── payment.js       # Атомарная транзакция оплаты заказа
│   ├── seed.js          # Заполнение БД демо-данными (запустить один раз)
│   └── global.css       # Глобальные CSS-переменные и утилиты
├── client-lk/           # Личный кабинет пользователя
├── admin/               # Админ-панель (пользователи, меню, отчёты)
├── kitchen-terminal/    # Кухонный терминал (список на готовку)
├── delivery-terminal/   # Терминал выдачи (QR, отметка выдачи)
└── queue-screen/        # Экран очереди (Готовится / Готово)
```

## Коллекции Firestore

| Коллекция      | Назначение                                     |
|----------------|------------------------------------------------|
| `users`        | Пользователи, роли, баланс клиентов            |
| `items`        | Справочник блюд / меню                         |
| `orders`       | Производственная сущность заказа               |
| `checks`       | Финансовый чек (создаётся только после оплаты) |
| `transactions` | Финансовые проводки (баланс / карта)           |

## Инициализация демо-данных

После первого запуска вызовите один раз из консоли браузера:

```js
import { seedDatabase } from '/shared/seed.js';
await seedDatabase();
```

## Роли пользователей

| Роль       | Доступ                                        |
|------------|-----------------------------------------------|
| `client`   | Личный кабинет: меню, заказ, история, баланс  |
| `cook`     | Кухонный терминал                             |
| `cashier`  | Терминал выдачи                               |
| `manager`  | Отчёты, просмотр заказов                      |
| `admin`    | Полный доступ: пользователи, меню, отчёты     |
